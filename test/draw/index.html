— 活动规则 —

一、活动时间:
参与时间:11月13日-11月24日;
开抢时间:11月22日-11月24日,每天 10:00-11:00 和 16:00-17:00。
1. 用户在开抢时间段内,通过活动页面点击“马上抢”,即可参与抢夺活动;
2. 活动期间如有部分奖品显示“剩余数量为0”,您可以在下一个开抢时间段刷新活动页面查看更新;
3. 银联商务APP注册用户可在活动页面进行抢夺活动,本次抢夺并非100%中奖,会有[没抢到]出现,请知悉。

二、抢夺活动规则
1. 活动期间，用户首次进入活动页面参与活动，即可获得1次抢夺机会;
2. 活动期间，用户首次点击“立即邀请”并分享成功后，可额外获得1次抢夺机会;
3. 用户通过“立即邀请”分享页面，每成功邀请1位新用户注册“银联商务”APP，可额外获得1次抢夺机会;
4.一旦发现作弊行为，银联商务APP有权取消您的获奖资格。

三、奖品规则
1. 活动奖品:
[iphone X 64G]每个抢夺时段送出2台;
[10元手机充值券]每个抢夺时段送出3000张;
[5元手机充值券]每个抢夺时段送出14000张;
2.手机充值券使用条件:请在有效期内登录银联商务APP使用,仅可给注册手机号充值；可以在“活动详情页-奖品明细”点击立即使用或点击APP“我的-我的优惠券”查看；在“我的-手机充值”进行使用;
3.iPhone X 64G (国行版)产品将在正式开售后60个工作日内发出,若正规市场渠道无法正常供应该产品,则会提供相同价值奖品,特殊情况另行通知(个人所得税需自理)。礼品来源官方授权供应商,暂不提供发票和售后服务,奖品售后问题请自行联络产品厂商咨询;
4. 实物中奖名单将于2017年12月15日前在银联商务官方微信公众号中公布;
5. 实物中奖后请您在2017年11月25日之前填写收货信息,并确保所填地址为有效邮寄收件地址。如因地址或手机号码错误所产生的一切不利后果,由用户本人承担。逾期不填将视为自愿放弃实物奖励。
